# Elevate_Lab_WD_1
This project is a simple and clean responsive landing page built using HTML5 and CSS3. The goal of this project is to understand the fundamentals of web development, including layout design, styling, and responsiveness.

Objective
1. Create a responsive landing page with:
2. Header (logo and navigation)
3. Hero section (main content with call-to-action)
4. Footer (basic information or social links)
5. Learn how to use Flexbox, Grid, and Media Queries

Tools: VS Code (free), Chrome Browser, live-server extension.
Deliverables: HTML and CSS files rendering a responsive landing page.

Features
1. Responsive design for mobile and desktop screens
2. Clean and simple UI
3. Navigation bar with links
4. Hero section with heading, text, and button
5. Footer section
